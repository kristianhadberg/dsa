Enter an array of numbers in this format: 5, 4, 5, 1... and click start!
Change tickrate, default is 500ms

note: you might need to zoom out a little in order to see the entire visualisation simultaneously

Input array

1. Loop through the input array, count each appearance of a number and insert it into the 'count array'

Count array

Cumulative array

2. Change the count array into a 'cumulative sum' array. This is done by looping through the array, starting at index 1, and adding the value of the element at the previous index to value of the element at the current index ( arr[i] = arr[i - 1] + arr[i] ). This array is now used to keep track of which indexes the elements belong at in the final array.

Output array

3. Loop through the input array in reverse order to keep the sorting stable. Insert the value in the output array by checking which index it fits in the cumulative array and decrement the cumulative array.
